# Sounds

Place your `.mp3` sound files here. Expected filenames:

| File | Trigger |
|------|---------|
| `linear_start.mp3` | Linear regression / logistic regression starts |
| `forest_start.mp3` | Random forest starts |
| `neural_start.mp3` | Neural network epoch detected |
| `neural_complete.mp3` | Neural network training complete |
| `cluster_start.mp3` | K-Means / clustering starts |
| `cluster_complete.mp3` | Clustering converged |
| `svm_start.mp3` | SVM starts |
| `boost_start.mp3` | Gradient boosting (XGBoost, LightGBM, CatBoost) starts |
| `pca_start.mp3` | PCA / UMAP / t-SNE starts |
| `complete.mp3` | Generic completion fallback |

Free sound sources: [freesound.org](https://freesound.org), [mixkit.co](https://mixkit.co/free-sound-effects/)
